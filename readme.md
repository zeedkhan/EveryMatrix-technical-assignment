# Installation
Backened: Express JS
Frontend: Vite JS
Dev: Docker
Database: Postgresql


# Installation

```bash
git clone https://github.com/zeedkhan/EveryMatrix-technical-assignment.git
```

# Add environment
copy ```.env.example``` and rename to ```.env```
replace the value

# Start application
```bash
docker compose build
docker compose up
```
