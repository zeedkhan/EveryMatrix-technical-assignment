## Install

Install

```bash
npm install
```

# Add environment
copy ```.env.example``` and rename to ```.env```
replace the value


# Development
Prisma ORM (database)
```bash
npx prisma generate
npx prisma db push
```

# Start application
```bash
npm run build
npm run start
```
